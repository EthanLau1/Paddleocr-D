# Paddleocr-D — 浏览器里用的 OCR

## 用就完了

```bash
ocr-start   # 启动 → 浏览器自动打开 → 拖图/PDF 识别
ocr-stop    # 用完关掉
ocr-open    # 重新打开浏览器页面
```

## 功能

| 功能 | 说明 |
|------|------|
| 📸 图片识别 | JPG / PNG / BMP / TIFF |
| 📄 PDF 识别 | 自动转图片，逐页 OCR |
| 🌐 自动语言 | 预加载中文秒出，置信度不够自动换其他 10 种语言 |
| 📝 Markdown 输出 | 按 Y 坐标分段落，置信度表格 |
| ⚡ 一键启动 | `Start.command` 双击就跑，后台守护 |

## 文件说明

| 文件 | 干什么的 |
|------|---------|
| `ocr_ui.py` | Web UI 主程序（Gradio） |
| `Start.command` | 一键启动（后台运行 + 自动打开浏览器） |
| `Stop.command` | 一键停止 |
| `venv/` | Python 虚拟环境 |
| `images/` | 测试图片目录 |
